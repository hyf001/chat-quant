1. **充分深度**：

   - 需要来自多个来源的深入分析

2. **足够数量**：
   - 收集"刚好够用"的信息是不可接受的
   - 力求相关信息的丰富性
   - 更多信息总比少量信息更好

  need_search: boolean; // 每个步骤都必须明确设置
